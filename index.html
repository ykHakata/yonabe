よなべperlとは

  • 現代社会においてあらゆる分野でコンピューターが活用され
    そのコンピューターを制御してるのはプログラミングのチカラです。
  • インターネットの普及で
    プログラミングスキルの必要性が一層高まってきていると同時に
    プログラミングスキルの習得への障害も昔に比べれば低くなったものの、
    未経験者からのスキル習得は一般的に容易くなったとは言いがたい状況です。
  • 未経験者からIT企業への就労、フリーランス希望の方までも対象にし
    現実的なアドバイスをしながら、
    webアプリケーション開発を通じてプログラミングスキルアップをしてゆきます。
  • よなべperlは才能あるITエンジニアの育成と発掘を目指します。

よなべperl３つの特徴

  • ・webアプリケーションという成果物の完成を目指す
  • ・基礎技術習得には解説ありきでなく、問題を解いてゆく事により体得と目指す
  • ・現役のITエンジニアがアドバイスをする

Webシステムの概要

webシステムとはあらゆる端末からあらゆる場所より双方向で通信できる仕組み

web サーバー

  • パソコン (Windows, Mac, Linux)
  • スマートフォン (iPnone, Android)
  • タブレット端末 (iPad, Android)
  • その他webブラウザが使える端末

簡単にいうと

  • ●インターネットとは、webサーバaーとwebブラウザを搭載した端末で通信する仕組み
  • ●世界中のあらゆる場所から、複数の端末で通信可能
  • ●既存の閉鎖されたコンピューターシステムはwebシステムに移行する事が予測される

―システムを支えるプログラミングのチカラ―

システムとは仕組み、プログラミングとは仕組み化できる能力

  • ・システムとは生活のあらゆる分野に存在している
  • ・ネットオークションサイト、電車の自動改札、自動車のブレーキシステムなど
  • ・システムを構築しているのはプログラミングのチカラ

Webの世界を体験(オンラインコース)

webサーバー、webブラウザの関係性と概要を体感する

  • ・〈windows・Mac対応〉
  • ・HTML CSS(WEB画面作成の基本)
  • ・HTML CSS JavaScript(動きのある画面の作成)
  • ・WEBサーバー(静的なHTMLファイルをアップロード)

必須スキルを個別に習得(オンラインコース)

プログラミングを体で覚える、必要に応じて個別にアドバイス

  • ・〈Macのみ対応〉
  • ・テキストエディッタ(プログラミング用エディッタ SublimeTextを使ってテキスト加工)
  • ・UNIX/bash(シェルの使い方を通じてUNIXシステムに慣れる)
  • ・SSH/Git/GitHub(分散型バージョン管理システムと秘密鍵と公開鍵の基本)
  • ・Perl(現代的なモダンPerlの基本的なコーディング)
  • ・Perl/Mojolicious(WEBフレームワークについて)
  • ・SQL(SQLiteでリレーショナルデータベースの基本を知る)
  • ・WEBサーバー(さくらVPSの最低限度の設定を通じてサーバーを知る)

Webアプリを作成(オンラインコース)

webアプリケーションという成果物を作成し、第三者にプレゼンテーション

  • ・IT企業などでのプログラミングに携わる業務を行う事を前提とします。
  • ・IT企業就労の未経験者は成果物を持ってして企業側へアプローチするのが現実的
  • ・webアプリの企画、仕様定義、実装、プレゼンまでをする事で総合的なチカラをつける

プログラマーとの交流(リアル勉強会)

上級者との出会い、さらなる可能性と高みを目指す

  • ・初級者から上級者までリアルに交流する事により、スキル以外の考え方なども習得
  • ・IT業界の実情を知り、自分自信にあった道を目指す
  • ・IT企業への就労や独立へのネットワークづくりに役立てる
