よなべ Perl とは

現代社会においてあらゆる分野でコンピューターが活用され、
そのコンピューターを制御してるのは プログラミングのチカラです。

インターネットの普及でプログラミングスキルの必要性が
一層高まってきていると同時にプログラミングスキルの習得への障害も
昔に比べれば低くなったものの、未経験者からのスキル習得は
一般的に容易くなったとは言いがたい状況です。

未経験者から IT 企業への就労、フリーランス希望の方までも対象にし、
現実的なアドバイスをしながら、web アプリケーション開発を通じて
プログラミングスキルアップをしてゆきます。
よなべ Perl は才能ある IT エンジニアの育成と発掘を目指します。

よなべ Perl ３つの特徴

  1. webアプリケーションという成果物の完成を目指す
  2. 基礎技術習得には解説ありきでなく、問題を解いてゆく事により体得と目指す
  3. 現役のITエンジニアがアドバイスをする

Web システムの概要

Web システムとはあらゆる端末からあらゆる場所より双方向で通信できる仕組み

  • web サーバー -> パソコン (Windows, Mac, Linux)
  • web サーバー -> スマートフォン (iPnone, Android)
  • web サーバー -> タブレット端末 (iPad, Android)
  • web サーバー -> その他webブラウザが使える端末

簡単にいうと

  • インターネットとは、webサーバーとwebブラウザを搭載した端末で通信する仕組み
  • 世界中のあらゆる場所から、複数の端末で通信可能
  • 既存の閉鎖されたコンピューターシステムはwebシステムに移行する事が予測される

システムを支えるプログラミングのチカラ

システムとは仕組み、プログラミングとは仕組み化できる能力

  • システムとは生活のあらゆる分野に存在している
  • ネットオークションサイト、電車の自動改札、自動車のブレーキシステムなど
  • システムを構築しているのはプログラミングのチカラ

Web の世界を体験 (オンラインコース)

Web サーバー、Web ブラウザの関係性と概要を体感する

  • 〈windows・Mac対応〉
  • HTML CSS(WEB画面作成の基本)
  • HTML CSS JavaScript(動きのある画面の作成)
  • WEBサーバー(静的なHTMLファイルをアップロード)

必須スキルを個別に習得 (オンラインコース)

プログラミングを体で覚える、必要に応じて個別にアドバイス

  • 〈Macのみ対応〉
  • テキストエディッタ(プログラミング用エディッタ SublimeTextを使ってテキスト加工)
  • UNIX/bash(シェルの使い方を通じてUNIXシステムに慣れる)
  • SSH/Git/GitHub(分散型バージョン管理システムと秘密鍵と公開鍵の基本)
  • Perl(現代的なモダンPerlの基本的なコーディング)
  • Perl/Mojolicious(WEBフレームワークについて)
  • SQL(SQLiteでリレーショナルデータベースの基本を知る)
  • WEBサーバー(さくらVPSの最低限度の設定を通じてサーバーを知る)

Web アプリを作成 (オンラインコース)

Web アプリケーションという成果物を作成し、第三者にプレゼンテーション

  • IT企業などでのプログラミングに携わる業務を行う事を前提とします。
  • IT企業就労の未経験者は成果物を持ってして企業側へアプローチするのが現実的
  • Web アプリの企画、仕様定義、実装、プレゼンまでをする事で総合的なチカラをつける

プログラマーとの交流 (リアル勉強会)

上級者との出会い、さらなる可能性と高みを目指す

  • 初級者から上級者までリアルに交流する事により、スキル以外の考え方なども習得
  • IT 業界の実情を知り、自分自信にあった道を目指す
  • IT 企業への就労や独立へのネットワークづくりに役立てる
